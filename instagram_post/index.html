Troubleshooting 400 (Bad Request) error:
1. Make sure token includes all required permissions (see top field)
2. Test token to auto-fill IG Account ID
3. Ensure Instagram account is linked to Facebook Page (Instagram Settings → Linked Accounts → Confirm Connection)
4. Use imgBB for image hosting (always works)
5. Regenerate token if error persists: https://developers.facebook.com/tools/accesstoken/
